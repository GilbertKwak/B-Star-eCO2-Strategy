## 3.2 PCHE 열설계

PCHE(Printed Circuit Heat Exchanger)는 SS316L 또는 Inconel 718 소재를 화학 식각하여 미세 채널(직경 0.5~2mm)을 형성합니다. 핀치포인트 ΔT≤5°C 달성을 위해 NTU-ε 방법으로 설계합니다.

### 3.2-1 설계 파라미터

| 파라미터 | 값 | 단위 |
|---|---|---|
| 열교환 효율 (ε) | ≥90 | % |
| 핀치포인트 (ΔT_pp) | ≤5 | °C |
| 설계 압력 | 130 | bar |
| 운전 압력 | 107 | bar |
| 소재 (고압측) | Inconel 718 | — |
| 소재 (저압측) | SS316L | — |
| 채널 직경 | 0.8 | mm |
| 단위 면적당 열유속 | 15~25 | MW/m² |

## 3.3 T-Q 선도 (GPU CDU 85°C 기준)

열원(GPU 냉각수): 85°C → 45°C 방열
CO₂ 작동유체: 32°C(흡입) → 67°C(PCHE HP 출구)
핀치포인트: 열원 45°C / CO₂ 40°C → ΔT = 5°C ✅
